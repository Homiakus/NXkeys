# Экспортированные роли Siemens NX 2512

Каталог предназначен только для проверенных `.mtx`, экспортированных из целевой версии Siemens NX 2512.

## Роль в адаптивной архитектуре

NXKeys не зависит от конкретного расположения команд в ribbon роли. Рабочие операции выполняются по точным `BUTTON ID`, а набор Leader определяется текущим приложением NX через Command Bridge.

Роль всё ещё влияет на:

- наличие лицензированных приложений;
- доступность и чувствительность UI-команд;
- локальные настройки выбора;
- видимость системных элементов;
- фактический `application_id` и `module_id`.

## Правила

- NXKeys не редактирует бинарный `.mtx`.
- Роль создаётся и экспортируется средствами NX.
- Копируется только целый проверенный файл.
- Установка роли по умолчанию отключена.
- `source_mtx` и `target_directory` задаются явно.
- Перед распространением проверяются все 14 модульных наборов, доступные по лицензии.
- Несовпадение роли не должно приводить к выполнению команды другого модуля.

## Рекомендуемое имя

```text
NX_Adaptive_Modules_2512.6000.mtx
```

## Проверка

После включения роли:

1. запустите NX через managed launcher;
2. откройте последовательно доступные приложения;
3. проверьте `context.json`;
4. убедитесь, что HUD показывает правильный модуль;
5. проверьте восемь команд без изменения производственных данных;
6. отдельно подтвердите destructive-команды на тестовой детали.
